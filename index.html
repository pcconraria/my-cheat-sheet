Cheat Sheet of HTML Made By Myself

This is a sheet with some elementes of HTML which I've elaborated with the help of CodeCademy.

Elements

Element Name Description
<a> Anchor The <a> element, also known as the “anchor” element, creates a link to another page or to a location in the current page. It can also be used to link to email addresses, phone numbers, and files.
<audio> Audio Represents an interface for adding audio content to the page.
<br> Break The <br> element represents a break in text. It is used when text needs to span multiple lines rather than being in-line, such as in an address.
<div> Division The <div> element represents a generic division of content. It has no semantic meaning, but will separate its contents from the rest of the document.
<h1>-<h6> Headings The <h_> elements represent text headings for sections of content. They range from <h1> being the highest level of heading to <h6> being the lowest.
